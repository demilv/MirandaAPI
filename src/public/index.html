API Documentation

Welcome to my API documentation. These are the directions u can use in your browser to check all the data

  • /rooms: Get a list of all rooms.
  • /users: Get a list of all users.
  • /reviews: Get a list of all reviews.
  • /bookings: Get a list of all bookings.

To get a specific item, use one of the previous endpoints followed by a bar and number; for example:

  • /users/1: Get the user with ID 1.
